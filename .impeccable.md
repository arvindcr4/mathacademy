## Design Context

### Users
MathAcademy is for serious self-directed learners studying math-heavy subjects. They use it in focused sessions and need the interface to make progress, difficulty, and next actions legible immediately.

### Brand Personality
Precise, motivating, intelligent.

### Aesthetic Direction
Dark study-cockpit UI with warm gold used for reward states, cool blue used for system guidance, and restrained celebratory motion. Typography should feel deliberate rather than default: a characterful display face for hierarchy and a clean sans for dense product UI.

### Design Principles
1. Show progress clearly and often.
2. Keep the base interface calm; let reward states carry the energy.
3. Avoid generic AI-product styling, purple bias, and over-nested card layouts.
4. Use motion for feedback and orientation, not decoration.
5. Treat accessibility features as part of the design system, not cleanup work.
