<div class="pilwrap" id="key-insights">
  <h2>
    <a href="#key-insights" name="key-insights" class="pilcrow"></a>
Key Insights
  </h2>
</div>
<ul>
<li>The curriculum structure uses a hierarchical Course → Topic → KnowledgePoint model</li>
<li>Knowledge point IDs use prefixes: <code>ci-kp-*</code> (coding interview), <code>sd-kp-*</code> (system design), <code>dp-kp-*</code> (design patterns), <code>rl-app-kp-*</code> (RL applications)</li>
<li>Each knowledge point needs: id, slug, and name fields</li>
<li>The page.tsx component renders content from an exampleProblems dictionary keyed by knowledge point slug</li>
</ul>
